# NomadCircle---hackathon
NomadCircle is a virtual tourism and travel companion platform that allows users to explore destinations through VR/360° views, get AI-powered travel guidance, and connect with like-minded travelers for safe and confident trip planning. Developed for Innovathon 1.0@JU Hackathon 2026.

 # NomadCircle – Virtual Travel & Travel Buddy Platform

Category: Tourism & AI  
Team: NomadCircle  
Institute: University of Jammu  

NomadCircle is a virtual tourism platform that helps users explore destinations virtually, get AI-based guidance, and connect with like-minded travelers.

---

## Problem Statement

Tourists depend on static images and biased reviews.
There is no realistic way to experience a place before visiting.
Many travelers feel unsafe and lack suitable companions.

---

## Solution

NomadCircle provides:
- Virtual destination exploration
- AI travel assistant
- Social travel matching

Core Idea:
Travel virtually first, decide confidently, and never travel alone.

---

## Key Features

- 360° VR Tours
- AI Chat Assistant
- Travel Buddy Matching
- Group Chat
- Personalized Recommendations

---

## How It Works

1. Create profile  
2. Select destination  
3. Explore virtually  
4. Use AI assistant  
5. Connect with travelers  
6. Plan trip  

---

## Technology Stack

Frontend: React / Flutter  
Backend: Node.js / Python  
AI: NLP Recommendation System  
Database: MongoDB / Firebase
